# Portfolio Project 3

## Introduction

Repository for the frontend of Portfolio Project 3

## Setup & Run Instructions

### Prerequisites

- Node.js (v20 or later)
- npm (v10 or later)

Follow these steps to run the project locally.

1. Clone the repository

```bash
git clone https://github.com/cit09-ruc-2025/portfolio-3.git
```

2. Navigate to the directory

```bash
cd portfolio-3
```

3. Copy the example environment file to `.env`:

   ```bash
   cp .env.example .env
   ```

> Update the `.env` file with your backend URL. Make sure to replace the placeholders with your actual backend URL.

4. Run the application:

   ```bash
    npm run dev
   ```
